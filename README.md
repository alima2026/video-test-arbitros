# Video Test Árbitros - Configurable V8

Esta versión corrige dos puntos importantes:

1. No trae respuestas inventadas.
2. Si el Excel de resultados está abierto, no se rompe: guarda una copia nueva con fecha y hora.

## Cómo usar

1. Colocar `video_1.mp4` hasta `video_11.mp4` en la carpeta `videos`.
2. Ejecutar `run_app.bat`.
3. Entrar a `Administrador`.
4. Clave: `admin123`.
5. Cargar las respuestas correctas reales:
   - decision_correcta
   - sancion_correcta
   - criterio_admin
   - explicacion_admin
6. Guardar configuración.
7. Recién ahí los árbitros podrán hacer el test.

## Opciones del participante

Decisión técnica:
- No falta
- Tiro libre directo
- Tiro libre indirecto
- Penal

Sanción disciplinaria:
- No tarjeta
- Amonestación
- Expulsión

## Total

11 videos x 4 puntos = 44 puntos.
